# BattleType

## Getting Started

### Requirements

* PostgreSQL
* Ruby 2.3.1

### Installation

```
$ bundle install
$ cp config/application.yml.sample config/application.yml
$ rails db:create
$ rails db:migrate
$ rake db:seed
$ rake dico:update
```

### Usage

Just launch the Rails app.

```
$ rails s
```
